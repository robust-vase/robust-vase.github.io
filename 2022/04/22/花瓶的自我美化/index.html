# 搭建hexo博客

参考视频

严格按照视频的来没有问题

  1. cnpm install -g hexo 注：管理员身份打开

  2. hexo init

  3. hexo s

  4. cnpm install hexo-deployer-git --save

  5. hexo g

  6. hexo d（如果不成功多试几遍）

注意： 若本地化打开不成功，切换各种各样的端口

  • hexo s -p 80

  • hexo s -p 5000

注意：若hexo d 之后没有正常上传，按Ctrl+ F5 刷新浏览器
